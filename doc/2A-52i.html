2A-52i: Rain Existence

Synopsis

2A-52i [-v] [-O] [-S] [-a [-s satellite_coinc_dir] ] rrmap_hdf_file re_ascii_file 
       BEGIN/MIDDLE/END

Description

Rain existence. 2A-52i produces rain percentage for each VOS from one granule. In addition, it specifies which VOS was obtained when the satellite was the closest to the radar during coincidence.

This program will create a new output file and will write header and rain existence information to it if the specified flag is BEGIN; it will append the rain existence information to the existing product file if the specified flag is MIDDLE/END.

Input Files

  1. rrmap_hdf_file - Hierarchical Data Format (HDF) file containing rain rate maps for one granule (2A-53 products).
  2. $GVS_DATA_PATH/sat_coinc_data/CT.yymmdd. See option -s below.

Temporary File

[/tmp/]2A52_[unsorted_file | *]

where, '*', is a widecard.

Output File

re_ascii_file - Ascii file containing rain percentage for each VOS from one granule; file may contain no VOS entry because of empty granule. The product may contain existence data of one site for one month. (This is the result of calling this program many times using the same output filename).

Options

2A-52i recognizes the following options:

-v: Output steps of program execution.

-O: Order the product file by VOS' date and time and remove duplicated entries before appending. Default: Do nothing.

-S: Processing without satellite coincident information lookup.

-a: Use ASCII satellite coincident file(s). Specify this for running in non-TSDIS environment. Default: Use TSDIS' database. Since satellite coincidence file contains data for one day, two satelliet coincidence files may be required for VOS having time + 30 minutes occurs on the next day. of time - 30 minutes.

-s: Specify directory that contains satellite coincidence files (CT.yymmdd). Default: $GVS_DATA_PATH/sat_coinc_data or /usr/local/trmm/data/sat_coinc_data if $GVS_DATA_PATH is not set.  This options only applies when -a is specified.


Return value

2A-52i returns zero for successful completion; -1 for failure; and -2 for interruption.
