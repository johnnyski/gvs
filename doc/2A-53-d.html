2A-53-d: Rain Rate Maps

Synopsis

2A-53 [-v] [options_list] 2A-53-d qc_ghdf_file csmap_ghdf_file rrmap_ghdf_file

singlevos_2A-53-d [-v] [options_list] qc_uf_file csmap_shdf_file rrmap_shdf_file

where:

options_list := [-s {firstfile|site}] [-b HBASE] [-f HFREEZE] [-n DBZNOISE] 
        [-p PRADAR] [-t TRADAR] [-d DEWPOINT] [-P DATAPATH] [-z ZRFILE] 
        [-g GAGFILE] [-w GWINFILE] [-k XKMP] [-a ALPHA] [-m MPONLY]

Description

Radar site rain rate map. 2A-53 calls singlevos_2A-53-d which converts reflectivities of a single VOS for 1C-51 product to rain rates. It creates a 2 km x 2 km CARPI (Constant Altitude Rectangular Position Image) to 200 km from radar. 2A-53 produces rain rate maps for one hour-granule.

Options

2A-53 and singlevos_2A-53-d recognize the following options:

-v Prints arguments and status of the program execution.
-s {firstfile|site} First file or site name.
-b HBASE Altitude for reflectivity map, to be used for rainfall. (Default: 2.0)
-f HFREEZE Altitude of 0 C isotherm. (Default: 5.0)
-n DBZNOISE Minimum reflectivity to process [dBZ]. (Default: 0.0)
-p PRADAR Surface pressure at the radar [mb]. (Default: 1000.0)
-t TRADAR Surface temperature at the radar [degrees kelvin]. (Default: 303.0)
-d DEWPOINT Surface dew point at the radar [degrees kelvin]. (Default: 295.0)
-P DATAPATH Path name where ZRFILE and GAGFILE are located. (Default: $GVS_DATA_PATH)
-z ZRFILE tring with the Ze-R input file name. (Default: zr_mlb.out (Melbourne) zr_drw.out (Lassen))
-g GAGFILE String with the Gauge coordinates file name. (Default: sitelist_cscale)
-w GWINFILE String with the gauge windows output file name. (Default: gwin.out)
-k XKMP Optional K for a power law Z-R. (Default: 0.0)
-a ALPHA Optional ALPHA for a power law Z-R. (Default: 0.0)
-m MPONLY Flag for WPMM (0), or a power-law Z-R (1). (Default: 0.0)

Input Files

  1. qc_uf_file - Quality controlled reflectivities for a single VOS in Universal Format (1C-51 product).
  2. qc_ghdf_file - Quality controlled reflectivities for one hour-granule in Hierarchical Data Format (HDF) (1C-51 products).
  3. csmap_hdf_file - Convective/stratiform map for a single VOS in HDF (2A-54 product).
  4. csmap_ghdf_file - Convective/stratiform maps for one hour-granule in HDF (2A-54 products).
  5. sitelist_cscale (Default for GAGFILE)--Lookup table.
  6. zr_{drw|mlb}.out (Default for ZRFILE)--Lookup table.

Temporary Files

The following file(s) are removed upon completion by the program.

  1. tmp<process_id>/<YYMMDD>.<granule#>.<radar_site>.<version#>.<HHMM>.uf.gz
  2. tmp<process_id>/<csmap_ghdf_file prefix>.HHMM.hdf
  3. tmp<process_id>/<uf file prefix>..hdf.2A-53-d
  4. csmap.ascii.tmp

Output Files

  1. rrmap_shdf_file - Rain rate map for a single VOS in a HDF file. Map consists of a 151x151 pixel image, with radar located at 76x76. Resolution is 2 km/pixel, thus data extends 150 km radius from the radar.
  2. rrmap_ghdf_file - Rain rate maps for one hour-granule in a HDF file.
  3. gwin.out (Default for GWINFILE).

Return value

2A-53 returns 0 upon successful completion; 1 upon successful processing for some VOSs of granule; 2 upon interruption; non-zero, otherwise.

singlevos_2A-53-d returns 0 upon successful completion; -2 upon interruption; -1 upon failure.
