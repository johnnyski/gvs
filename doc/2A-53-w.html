2A-53-w: Rain Rate Maps

Synopsis

2A-53 [-v] [options_list] 2A-53-w qc_ghdf_file csmap_ghdf_file rrmap_ghdf_file
singlevos_2A-53-w [-v] [options_list] qc_uf_file csmap_shdf_file rrmap_shdf_file

where:
options_list := [-S site_name] [-P pathname_of_lookup_tables] 
                        [-p parameters_file]

Description

Radar site rain rate map. 2A-53 calls singlevos_2A-53-w which converts reflectivities of a single VOS for 1C-51 product to rain rates. It applies polar-cartesian transformation with rain rate field and produces a 2 km x 2 km CARPI (Constant Altitude Rectangular Position Image) to 200 km from radar. 2A-53 produces rain rate maps for one hour-granule.

Options

2A-53 and singlevos_2A-53-w recognize the following options:

-v Prints arguments and status of the program execution.

-S site_name Specify site name. It is case insensitive and can either be a long or 4-digit name. See file, $GVS_DATA_PATH/sprint_radar_sites, for a list of sites. Default: Will obtain from the UF file.

-P pathname_of_lookup_tables Pathname of lookup tables. Value of the environment variable, GVS_DATA_PATH, or '/usr/local/trmm/data' when $GVS_DATA_PATH="" is used for default.

-p parameters_file File containing program's parameters. Default = <pathname_of_lookup_tables>/{melb|darw}_rainest.par

Input Files

  1. qc_uf_file - Quality controlled reflectivity for a single VOS in Universal Format (1C-51 product).
  2. qc_ghdf_file - Quality controlled reflectivity for one hour-granule in Hierarchical Data Format (HDF) (1C-51 products).
  3. csmap_shdf_file - Convective/stratiform map for a single VOS in HDF (2A-54 product).
  4. csmap_ghdf_file - Convective/stratiform map for one hour-granule in HDF (2A-54 products).
  5. <pathname_of_lookup_tables>/carttopol.{FL|DR} - Lookup tables.
  6. <pathname_of_lookup_tables>/poltocart.{FL|DR} - Lookup tables.
  7. <pathname_of_lookup_tables>/{melb|darw}_rainest.par - Lookup tables.

Note: Lookup tables, *.FL/melb and *.DR/darw, are used for Melbourne and Darwin data set, respectively.

Temporary Files

The following files are created and removed upon completion.

  1. tmp<process_id>/<YYMMDD>.<granule#>.<radar_site>.<version#>.<HHMM>.uf.gz
  2. tmp<process_id>/<csmap_ghdf_file prefix>.HHMM.hdf
  3. tmp<process_id>/<uf file prefix>.hdf.2A-53-w
  4. nrainest.<process_id>.tmp

Note: KMLB and DARW are for Melbourne and Darwin data set, respectively.

Output Files

  1. rrmap_shdf_file - Rain rate map for a single VOS in HDF file. Map consists of a 151x151 pixel image, with radar located at 76x76. Resolution is 2 km/pixel, thus data extends 150 km radius from the radar.
  2. rrmap_ghdf_file - Rain rate maps for one hour-granule in HDF file.

Return value

2A-53 returns 0 upon successful completion; 1 upon successful processing for some VOSs of granule; 2 upon interruption; non-zero, otherwise.

singlevos_2A-53-w returns 0 upon successful completion; -2 upon interruption; -1 upon failure.
