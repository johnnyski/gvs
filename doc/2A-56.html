2A-56: Rain Gauge

Synopsis

gmin_to_granule_ascii gmin1 [gmin2 [gmin3 ...]]

Description

This program granulates the input gmin files to monthly granule file(s).

Input Files

ASCII gmin file(s). Compressed file is acceptable.

Temporary File

None.

Output File

One or several ascii files having the name:

2A-56_site_network_gaugeID_YYYYMM_ver#.ascii
   where:
     site = MELB, DARW, KWAJ, HSTN, ...
     network = KSC, STJ, ...
     YYYY = Year (4 digit number)
     MM = Month (2 digit number)
     ver# = version (1)

Options

None.


Return value

Program returns zero for successful completion; -1 for failure.

Author
