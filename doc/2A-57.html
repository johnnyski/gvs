2A-57: Disdrometer

Synopsis

disdro_to_granule_ascii Disdro_file1 [Disdro_file2 [Disdro_file3...]]

Description

This program granulates the input disdrometer file(s) to monthly granule file(s).

Input Files

ASCII disdrometer file(s). Compressed file is acceptable.

Temporary File

None.

Output File

One or several ascii files having the name:

2A-57_site_network_gaugeID_YYYYMM_ver#.ascii
   where:
     site = MELB, DARW, KWAJ, HSTN, ...
     network = KSC, STJ, ...
     YYYY = Year (4 digit number)
     MM = Month (2 digit number)
     ver# = version (1)

Options

None.


Return value

Program returns zero for successful completion; -1 for failure.

Author
