uf2mudras: Convert UF to MUDRAS

Synopsis

uf2mudras [-v] [-S site_name] [-d YYMMDD] [-s sprint_input_template]
          [-V vertical_resolution] [-H horizontal_resolution]
          uffile outfile

Description

This program calls NCAR SPRINT interpolator to convert QCed UF file to pure MUDRAS file. Note: It removes files created by this script (e.g. "fort*", "fnt*", and "sprint.inp.<process id>") after it finished processing.

Options

uf2mudras recognizes the following options:

-v: Prints arguments and status of the program execution.
-S: Specify TSDIS site name. See file, $GVS_DATA_PATH/gv_radar_site_info.data for a list of sites. Default: Will obtain from the UF file..
-d: Date of radar data to be processed. For default, the date string from input filename (*<YYMMDD>.*) is used.
-s sprint_input_template: File contains SPRINT commands. This package provided two templates: 'sprint_entire_volume.inp' and 'sprint_base_scan.inp'. These templates are for interpolating the entire volume or just the base scan of the QCed UF file. For information on SPRINT commands' syntax, see NCAR SPRINT interpolator's documentation. Note: When creating your own template, you must do the following:

  • Use '10' for the Fortran logical unit number of the input file (P2 of 'INPUT' command) and '11' for the Fortran logical unit number of the output file (P2 of 'OUTPUT' command).
  • Fill P2 of the 'PROCESS' command with 'YYMMDD'.
  • Fill P6 of the 'INPUT' command with 'RADAR_NUM'.
  • Fill P6 of the 'GRID' command with 'HORIZONTAL_RES'.
  • Fill P9 of the 'GRID' command with 'VERTICAL_RES' if applicable.
This program will substitude those strings with the appropriate values. Default template = '$GVS_DATA_PATH/sprint_base_scan.inp' or '/usr/local/trmm/data/sprint_base_scan.inp' when $GVS_DATA_PATH = "").

-V: Specify vertical resolution in km. Default: 1.5.
-H: Specify horizontal resolution in km. Default: 2.0. 

Input

Quality controlled Universal Format (UF) file. 

Output

Mudras file.

Return value

uf2mudras returns 0 for successful completion; -1 for failure; -2 for interruption.
